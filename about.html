About The HADatAc Framework

What exactly is the HADatAc Framework?


What is HADatAc?

HADataC is a framework that enables combined collections of data and metadata in a way that metadata is properly and logically connected to data. HADatAc is built mainly for scientific research teams but can be used by anyone handling incoming data sets of physical measurements.

How is HADatAc useful?

HADatAc is useful because of the meaningful inferences that can be extracted from better understanding of the connection between the data and the context of that data. This rich metadata collection is then leveraged by the HADataC infrastructure to support the following: data management; data governance in terms of privacy, access and dissemination; uncertainty management; and (big) data analytics.

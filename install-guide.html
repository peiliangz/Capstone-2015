Installing HADatAc on Mac

Installing the required software:

Open the Terminal

Hold down Command and tap the Space key, type Terminal

If homebrew is not installed, you can install homebrew here: homebrew

If sbt is not installed, you can install sbt via:

Execute brew install sbt

If tar is not installed, you can install tar via:

Execute brew install tar

If wget is not installed, you can install wget via:

Execute brew install wget

If git is not installed, you can install git via:

Execute brew install git

If at least Java SDK 1.8+ is not installed, you can install Java here: Java

Check that your version of java is correct, execute javac -version (should be similar to javac 1.8.0_25)

If your version is not at least 1.8+:

Execute /usr/libexec/java_home -V then export JAVA_HOME=`/usr/libexec/java_home -v << java1.8 version from above >>` (Note: replace the text within << >> with your current java version output from javac -version)

Installing HADatAc:

Download the HADatAc project via:

Execute git clone https://github.com/paulopinheiro1234/hadatac.git

To install HADatAc:

Execute ./install_hadatac.sh

Go to the HADatAc directory:

Execute cd ~/hadatac

To run HADatAc:

Execute sbt run

HADatAc should be running now:

Check http://localhost:9000/hadatac/

If nothing is there, please report an issue here:

Installing HADatAc on Windows

Installing the required software:

If git is not installed, you can install git here: git-bash

If sbt is not installed, you can install sbt here: sbt tool

If at least Java SDK 1.8+ is not installed, you can install Java here: Java

To verify, execute the following command javac -version the output should be at least java 1.8+ (e.g. javac 1.8.0_25)

If wget is not installed, you can install wget here: wget

If WinRAR is not installed, you can install WinRAR here: WinRAR

Installing HADatAc:

For a 32 bit OS:

Type set path="C:\Program Files\WinRAR\";%path%

For a 64 bit OS:

Type set path="C:\Program Giles (x86)\WinRAR\";%path%

Navigate to the target directory:

Execute cd ~

Download the HADatAc project via the command line:

Execute git clone https://github.com/paulopinheiro1234/hadatac.git

Go to the HADatAc directory:

Execute cd ~/hadatac

Go to your browser and check:

Go to http://localhost:8983/solr and http://localhost:7574/solr

If nothing is there do the following to re-install:

Executecd C:\hadatac\solr\ run_solr4.bat restart then run_solr5.bat restart

To start HADatAc:

Execute sbt run

HADatAc should be running now:

Check http://localhost:9000/hadatac/

If nothing is there, please report an issue here:

Installing HADatAc on Mac

Mac OS X: Open `terminal` Install sbt --brew install stb Install tar --brew install tar Install wget --brew install wget 5. Install git --brew install git 6. Install Java SDK -- check the java version javac -version -- run the following command /usr/libexec/java_home -V --check the version and run export JAVA_HOME=`/usr/libexec/java_home -v << java1.8 version from above >> 7. Clone the HaDatac from the github git clone https://github.com/paulopinheiro1234/hadatac.git 8. Install HaDatac ./install_hadatac.sh 9. Running the HaDatac go to your your hadatac installation directory) and type the following command: sbt run 10. Open the browser and type: http://localhost:9000/hadatac/ 11. Wait for a few seconds for the page to render.

Installing HADatAc on Windows


Installing the required software

If git is not installed, you can install git here: git-bash

If sbt is not installed, you can install sbt here: sbt tool

If at least Java SDK 1.8+ is not installed, you can install Java here: Java

To verify, execute the following command javac -version the output should be at least java 1.8+ (e.g. javac 1.8.0_25)

If wget is not installed, you can install wget here: wget

If WinRAR is not installed, you can install WinRAR here: WinRAR

For a 32 bit OS:

Type set path="C:\Program Files\WinRAR\";%path%

For a 64 bit OS:

Type set path="C:\Program Giles (x86)\WinRAR\";%path%

Navigate to the target directory:

Execute cd ~

Install the HADatAc project via the command line:

Execute git clone https://github.com/paulopinheiro1234/hadatac.git

Go to the HADatAc directory:

Execute cd ~/hadatac

Go to your browser and check:

Go to http://localhost:8983/solr and http://localhost:7574/solr
  • Go to browser and check if both
  • http://localhost:8983/solr and http://localhost:7574/solr are running well
  • If not, execute command to re-install
  • cd c:\hadatac\solr\
  • run_solr4.bat restart
  • run_solr5.bat restart
  • Execute command sbt run to start the project
  • HADatac should be running at http://localhost:9000/hadatac/

Installing HADatAc on Mac

Mac OS X: Open `terminal` Install sbt --brew install stb Install tar --brew install tar Install wget --brew install wget 5. Install git --brew install git 6. Install Java SDK -- check the java version javac -version -- run the following command /usr/libexec/java_home -V --check the version and run export JAVA_HOME=`/usr/libexec/java_home -v << java1.8 version from above >> 7. Clone the HaDatac from the github git clone https://github.com/paulopinheiro1234/hadatac.git 8. Install HaDatac ./install_hadatac.sh 9. Running the HaDatac go to your your hadatac installation directory) and type the following command: sbt run 10. Open the browser and type: http://localhost:9000/hadatac/ 11. Wait for a few seconds for the page to render.

Installing HADatAc on Windows

1.Open the git bash
If you do not have git, you need to install:git-bash

2.Execute command “sbt”
If sbt is not installed, you need to install: sbt tool

3.Execute command “javac -version”
If program fails, you need to install:java

4.Execute command “wget”
If wget is not installed, you need to install:wget

5.Execute command “winrar”
If WinRAR is not installed, install: WinRAR Then for 32 bit operating system, type set path="C:\Program Files\WinRAR\";%path% Or for 64 bit operating system, type set path="C:\Program Files (x86)\WinRAR\";%path%

6.Go to target directory by executing “cd ~”

7.Execute command line to clone HADataC project

git clone https://github.com/paulopinheiro1234/hadatac.git

8.Go to HADataC directory by executing “cd ~/hadatac”

9.Execute command install_hadatac.bat to install HADataC

10.Go to browser and check if both
http://localhost:8983/solr and http://localhost:7574/solr are running well If not, execute command to re-install< cd c:\hadatac\solr\ run_solr4.bat restart run_solr5.bat restart

11.Execute command “sbt run” to start the project

12.HADatac should be running at “http://localhost:9000/hadatac/”
